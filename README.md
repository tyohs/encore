# 🎤 みんカラ - みんなで作る、みんなのカラオケ

カラオケを「聴く」から「参加する」へ。全員が主役になれる新感覚カラオケアプリ。

## ✨ 特徴

### 🎭 3つの役割
- **🎤 シンガー**: 歌詞表示で快適に歌唱、ファンサリクエストに応える
- **🎸 バンド**: 4種の楽器でリズムゲーム参加（ドラム/ギター/キーボード/ベース）
- **🔦 オーディエンス**: ペンライト＆コールで応援

### 📡 リアルタイム連携
- コール＆リクエストが歌手画面に即時表示
- カスタムメッセージがニコニコ風に流れる
- BroadcastChannel APIによるタブ間同期

### 🎮 スコア消費型コール
- リズムゲームでポイント獲得
- ポイントを使ってコール/リクエスト発動
- 4方向スワイプ対応（上下左右で異なるコール）

### 🎬 MV背景対応
- MP4アップロードでプロセカ風の没入体験
- オーディオと自動同期

### 💅 優れたUI/UX
- Glassmorphismデザイン
- 振動フィードバック
- スペシャルノートエフェクト

## 🚀 起動方法

```bash
npm install
npm run dev
```

http://localhost:3000 でアクセス

## 📝 デモ手順

1. ルームを作成
2. **2つのタブ**で同じルームに参加
3. タブ1: シンガーモードでプレイ開始
4. タブ2: バンドモードでプレイ
5. コール/リクエスト/メッセージを送信
6. → 歌手画面にリアルタイム表示！
