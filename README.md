# 🎤 みんカラ - みんなで作る、みんなのカラオケ

カラオケを「聴く」から「参加する」へ。全員が主役になれる新感覚カラオケアプリ。

## ✨ 特徴

- **🎤 シンガーモード**: 歌詞表示で快適に歌唱
- **🎸 バンドモード**: リズムゲームで演奏参加（ドラム/ギター/キーボード/ベース）
- **📡 リアルタイム連携**: コール＆リクエストが歌手画面に即時表示
- **🎬 MV背景**: MP4アップロードで没入体験
- **💅 Glassmorphism UI**: 洗練されたデザイン

## 🚀 起動方法

```bash
npm install
npm run dev
```

http://localhost:3000 でアクセス

## 🎵 収録曲

1. 仰げば尊し (BPM 96)
2. シャイニングスター (BPM 158)
3. Escort (BPM 80)
4. バーニングハート (BPM 142)

## 🛠️ 技術スタック

- Next.js 16 (App Router)
- TypeScript
- Tailwind CSS
- Framer Motion
- Zustand
- BroadcastChannel API

## 📱 使い方

1. ルームを作成
2. 役割を選択（シンガー / バンド / オーディエンス）
3. 曲を選んでプレイ開始！
4. バンドからのコール・リクエストがシンガーにリアルタイム表示
